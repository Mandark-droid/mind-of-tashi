# Bilingual mind-scroll lexicon — IAST transliteration + Devanagari + English gloss.
#
# Used in two places:
#   1. As an inline glossary inside prompts.build_system (so the teacher / SFT /
#      GRPO model has high-frequency Hindi/Sanskrit terms at hand when emitting
#      <think> blocks).
#   2. As a regex/lexicon source for the self-play quality filter — rounds whose
#      <think> contains zero hits against this file are flagged as English-only
#      drift and dropped.
#
# Format: <iast>\t<devanagari>\t<english_gloss>\t<notes>
# Comments start with '#'. Empty lines are ignored.
#
# Keep this small (~30 terms). The model should *taste* the register, not recite
# a dictionary. 3-6 woven terms per mind-scroll is the target density.

parvat	पर्वत	mountain	Mountain Stance flavour
patthar	पत्थर	stone	Hindi; immovable
